Description

It's no fun when the field has the same setup every time and you know where all the mines are located. Let's generate a new random configuration every time the user wants to play the game.

To improve the program, we need to let the user choose how many mines they want on the field. The user just needs to input a number with the keyboard.

Objectives

Your program should ask the user to define the number of mines on a 9x9 field with the message "How many mines do you want on the field? "; it will then use the input number to initialize the field and display it with the mines. Mines are still visible to the user, you will hide them later. 

Make sure to use the following marking symbols:

  • X for mines
  • . for safe cells

You can use Random.nextInt(...) to generate random numbers.

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.
